Steps to see my Javascript Output:

  • On Windows Laptop Or Desktop:

    1. Press Ctrl+Shift+I on keyboard.
    2. Click on Console tab.
    3. See the Output.

  • On Andriod Mobile:

    1. Open a New Tab.
    2. Click on the Star Icon to Bookmark it.
    3. Click on the Edit Bookmark at the bottom pop-up window.
    4. Edit the Name as "Inspect Element".
    5. Edit the URL with this code "javascript:( () => { var script = document.createElement('script'); script.src = "//cdn.jsdelivr.net/npm/eruda"; document.body.appendChild(script); script.onload = (() => eruda.init()); } )()".
    6. Open this page.
    7. Go on the search bar on top.
    8. Type "Inspect Element".
    9. Click on Inspect Element with star icon.
    10. Click on the Dev Tools icon (a hammer and a saw icon).
    11. Click on Console Tab.
    12. See the Output.
